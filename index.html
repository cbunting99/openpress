Quick Start Guide

Deploy your WordPress infrastructure in minutes

1 Clone Repository

git clone https://github.com/cbunting99/openpress.git cd openpress

2 Configure Environment

Edit docker-compose.yml and update the following variables:

DOMAIN=yourdomain.com EMAIL=admin@yourdomain.com

3 Deploy Infrastructure

docker-compose up -d

4 Verify Deployment

# Check service health curl http://yourdomain.com/health # View service logs docker-compose logs webserver
